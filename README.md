# 🏋️‍♂️ Physique Fitness LP Template

Next.js + Tailwind + shadcn/ui で構築した  
**フィットネス・ジム・美容サロン向け集客LPテンプレート**です。  
短納期でハイクオリティなLPを量産・カスタマイズできる構成になっています。

## ✨ 特徴

| 特徴 | 内容 |
|------|------|
| ⚡ **モダン構成** | Next.js 14 + Tailwind + shadcn/ui を採用 |
| 🧱 **コンポーネント分割** | Hero / Concept / Plans / Testimonials などを独立管理 |
| 📱 **スマホ最適化** | 全デバイス対応・高速レンダリング |
| 🪶 **デザイン拡張性** | 色・フォント・セクション差し替えで他業種にも対応可能 |
| 🔧 **CMS化も視野に設計** | PrismaやPlanetScale導入で動的化に拡張可能 |

## 🧠 カスタマイズ例

| 用途 | 修正箇所 | 所要時間目安 |
|------|-----------|---------------|
| 文言・画像差し替え | `/app/components/` | 約1時間 |
| 色味変更（テーマカラー） | `/globals.css` | 約30分 |
| 新セクション追加 | `/app/components/` に追加 | 約1〜2時間 |
